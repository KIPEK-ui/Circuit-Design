# Circuit-Design

This repository contains the work for **ICS 4111 [Apr-Jul 2026] Practical Exercise 2: Circuit Design** in the course **ICS 4111: Embedded Systems & IoT**.

## Overview

This practical exercise focuses on circuit design and analysis using software tools. The work is divided into two main parts: theoretical calculations and schematic diagram development.

## Part A: Circuit Calculations

For each of the given circuits, the following calculations are performed:

1. **Total Resistance** in ohms
2. **Total Current** from the power supply
3. **Current through each resistor**
4. **Voltage drop through each resistor**
5. **Power dissipated through each resistor** (using Watt's Law: Power = Voltage × Current)

The results of these calculations are documented in a PDF file.

## Part B: Schematic Diagrams

1. **Circuit Schematics**: Develop schematic diagrams for the two circuits described in Part A using EasyEDA (or equivalent software like KiCad). Each schematic includes a table listing all components.

2. **Arduino Nano with DHT22 Sensor**: Design a schematic diagram for an Arduino Nano microcontroller connected to a DHT22 temperature and humidity sensor. The schematic includes a table listing all components. The DHT22 requires a 3.3V-5V power supply.

All designs are exported as PDF documents.
